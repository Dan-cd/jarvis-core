## PASSO 3 — DEV_MODE.md (poder controlado)

### O que é Dev Mode

Dev Mode é um **estado explícito** que permite:

* criação de sistemas
* refatoração profunda
* decisões arquiteturais
* leitura e explicação de código

### Ativação

* Só por comando explícito do usuário
* Sempre confirmada pelo sistema
* Sempre registrada em contexto

### Regras do Dev Mode

* Nenhuma execução automática sem confirmação humana
* Toda sugestão vem com **trade-offs**
* Toda edição vem com **diff explicável**

### Fora do Dev Mode

* Sem criação de sistemas
* Sem decisões arquiteturais
* Sem execução sensível
